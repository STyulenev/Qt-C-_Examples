# Felgo-ListView-Example

## Описание

Пример использования AppListView в Felgo/QML на основе различных моделей:
- На основе ListModel (qml);
- На основе array (js);
- На основе SortFilterProxyModel (felgo).

## Сборка проекта

Проект необходимо собирать с помощью Felgo IDE.

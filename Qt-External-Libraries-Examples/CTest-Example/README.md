# CTest-Example

## Описание

Пример использования CTest для тестирование программы

## Сборка

### CMake

Установка Catch макросов для тестов:

```bash
sudo apt-get install catch
```

или скачать catch.hpp и вставить как часть проекта.

Сборка проекта:

```bash
cd build
cmake ..
make
```

Запуск тестов:

```bash
ctest
```

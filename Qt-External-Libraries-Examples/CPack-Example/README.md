# CPack-Example

## Описание

Пример автоматизированной сборки deb пакета с инструментом упаковки CPack

## Сборка
### CMake
 - Сборка проекта
```bash
mkdir build
cmake ..
make
```
 - Сборка deb пакета: имя_текущая.версия_архитектура.deb
```bash
cpack -G DEB
```
 - Проверка lintian на ошибки и предупреждения
```bash
lintian <deb_name>
```

# QListView-Example

## Описание

Пример использования виджета QListView-Example со сложной моделью, а также с делегатом и без него.

![alt text](doc/QListView-Example.png)

## Сборка проекта

Проект может быть собран из QtCreator или из папки build коммандами:
CMake:

```
cmake ..
make
```

QMake:

```
qmake ../-project
qmake ..
make
```
